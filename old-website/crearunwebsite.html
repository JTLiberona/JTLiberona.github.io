CÓMO CREAR TU PRIMER SITIO WEB EN HTML

Lo primero es entender que cada website esta compuesto por una estructura básica con una etiqueta de inicio <> y cierre de código </>.
La estructura del website esta compuesta por 

      Inicio del código HTML.
      La cabecera donde insertamos los metadatos que los motores de búsqueda indexarán.
      El cuerpo de la página web
      El cierre del código HTML.

En resumen esto quedará así

    
<html> --> Es la etiqueta de apretura para que el navegador interprete el código como HTML.
      
<head> --> Es la cabecera y representa el lugar donde ingresaremos los meta datos.
      
</head>--> Cierre de la etiqueta cabecera.
      
<body> --> Apertura del cuerpo de la página web.
      
</body> --> Cierre de la etiqueta cuerpo.
    
</html> --> Cierre de la etiqueta HTML.

TITULO o TITLE


A continuación definiremos el Título de la página.


  <titulo> INTRODUCIMOS EL TITULO </titulo>.

Para alinear podemos usar las siguientes etiquetas
  <left><title>INSERTAR EL TITULO</title></left>          para alinear el título a la izquierda.
  <center><title>INSERTAR EL TITULO</title></center>      para alinear el título al centro.
  <right><title>INSERTAR EL TITULO</title></right>        para alinear el título a la derecha.

CUERPO o BODY


En este lugar podemos introducir nuestro contenido:
  -El texto.
  -Separadores.
  -Enlances/links.
  -Fotos y videos.
  -Tablas.
  -Campos de registro o login.
  -etc...

EL TEXTO

El texto lo introducimos de manera normal, sin embargo si quisieramos que aparezca tal como lo hemos escrito en el editor de código deberemos poner la siguiente etiqueta.


  <pre>INSERTAR EL TEXTO A NUESTRO GUSTO, con saltos de línea etc </pre>.
  <h1>INSERTAR TEXTO</h1>      Con esta etiqueta insertamos un encabezado; a medida que aumentamos el número(hasta 6) disminuye el tamaño.

SEPARADORES

Para separar con una línea escribimos el siguiente código


  <hr style="border:15px;">         donde 15px será el grosor.
  <hr style="border-color:red;">    con esta etiqueta definimos color; como por ejemplo ROJO.
  <hr style="width:75%;">           acá definimos el ancho de la línea separadora.

ENLACES

El enlace se introduce con la siguiente etiqueta.


  <a href="ENLACE/URL/EMAIL>INTRODUCIR EL TEXTO DEL ENLACE"</a>

FOTOS y/o VIDEOS

Cada vez que quieras anexar una foto debes ingresar el siguiente código. 

    <scr="URL DE LA IMÁGEN">
